# Smart Locker - Arduino Project
IOT Device that uses fingerprint module and solenoids to set the security for the lockers. 

## Connections 
Connection from Fingerprint Module 1 to Arduino Uno

| Fingerprint Module  | Arduino |
| ------------- | ------------- |
| TX  | 1  |
| RX  | 2  |
| VCC  | 3.3V  |
| GND  | GND  |
